device

変換、送信対象の端末

hotentry

新着投稿だけをまとめたデータを作る

update.strong

更新漏れが無い様に改稿日の分は必ずDLする

update.convert-only-new-arrival

更新時に新着のみ変換を実行する

update.sort-by

アップデートを指定した項目順で行う

convert.copy-to

変換したらこのフォルダにコピーする
※注意：存在しないフォルダだとエラーになる

convert.no-open

変換時に保存フォルダを開かないようにする

convert.multi-device

複数の端末用に同時に変換する。deviceよりも優先される。
ただのEPUBを出力したい場合はepubを指定

convert.copy-to-grouping

copy-toで指定したフォルダの中で更にdevice毎にフォルダを振り分ける

send.without-freeze

一括送信時に凍結された小説は対象外にする

theme

WEB UI 用テーマ選択

hotentry.auto-mail

新着投稿作成時にメールを送る(mail設定済みの場合)

update.logging

更新時のログを保存する

convert.no-epub

EPUB変換を無効にする

convert.no-mobi

MOBI変換を無効にする

convert.no-strip

MOBIのstripを無効にする
※注意：KDP用のMOBIはstripしないでください

convert.no-zip

i文庫用のzipファイル作成を無効にする

convert.inspect

常に変換時に調査結果を表示する

download.interval

各話DL時に指定した秒数待機する。デフォルト0

download.wait-steps

この値で指定した話数ごとにウェイトを入れる
※注意：11以上を設定してもなろうの場合は10話ごとにウェイトが入ります

download.use-subdirectory

小説を一定数ごとにサブフォルダへ分けて保存する
※注意：小説を大量に同一フォルダに保存するとパフォーマンスが劣化する回避策

send.backup-bookmark

一括送信時に栞データを自動でバックアップする(KindlePW系用)

multiple-delimiter

--multiple指定時の区切り文字

Global な設定はユーザープロファイルに保存され、すべての narou コマンドで使われます

difftool

Diffで使うツールのパスを指定する
※WEB UIでは使われません

difftool.arg

difftoolで使う引数(指定しなければ単純に新旧ファイルを引数に呼び出す)
特殊な変数
%NEW : 最新データの差分用ファイルパス
%OLD : 古い方の差分用ファイルパス

no-color

コンソールのカラー表示を無効にする
※要WEB UIサーバ再起動

server-port

WEBサーバ起動時のポート

server-bind

WEBサーバのホスト制限(未設定時:起動PCのIP)

over18

18歳以上かどうか

dismiss-notice

お知らせを消す

default.* 系の設定は個別の変換設定で未設定の項目の挙動を指定することが出来ます

default.enable_inspect

小説に対する各種調査を実行する。結果を表示するには narou inspect コマンドを使用

default.enable_convert_num_to_kanji

数字の漢数字変換を有効にする

default.enable_kanji_num_with_units

漢数字変換した場合、千・万などに変換する

default.kanji_num_with_units_lower_digit_zero

〇(ゼロ)が最低この数字以上付いてないと千・万などをつける対象にしない

default.enable_alphabet_force_zenkaku

アルファベットを強制的に全角にする。falseの場合英文は半角、それ以外は全角になる

default.enable_half_indent_bracket

行頭かぎ括弧に二分アキを挿入する

default.enable_auto_indent

自動行頭字下げ機能。行頭字下げが行われているかを判断し、適切に行頭字下げをするか

default.enable_force_indent

行頭字下げを必ず行うか。enable_auto_indent の設定は無視される

default.enable_auto_join_in_brackets

かぎ括弧内自動連結を有効にする
例)
「～～～！
　＊＊＊？」 → 「～～～！　＊＊＊？」

default.enable_auto_join_line

行末が読点で終わっている部分を出来るだけ連結する

default.enable_enchant_midashi

［＃改ページ］直後の行に中見出しを付与する（テキストファイルを直接変換する場合のみの設定）

default.enable_author_comments

作者コメントを検出する（テキストファイルを直接変換する場合のみの設定）

default.enable_erase_introduction

前書きを削除する

default.enable_erase_postscript

後書きを削除する

default.enable_ruby

ルビ処理を有効にする

default.enable_illust

挿絵タグを有効にする（false なら削除）

default.enable_transform_fraction

○／×表記を×分の○表記に変換する。日付表記(10/23)と誤爆しやすいので注意

default.enable_transform_date

日付表記(20yy/mm/dd)を任意の形式(date_formatで指定)に変換する

default.date_format

書式は http://bit.ly/1m5e3w7 を参考

default.enable_convert_horizontal_ellipsis

中黒(・)を並べて三点リーダーもどきにしているのを三点リーダーに変換する

default.enable_convert_page_break

`to_page_break_threshold` で設定した個数以上連続する空行を改ページに変換する

default.to_page_break_threshold

ここで設定した値が `enable_convert_page_break` に反映される

default.enable_dakuten_font

濁点フォントを使用する。false の場合は縦中横による擬似表現を使用する

default.enable_display_end_of_book

小説の最後に本を読み終わった表示をする

default.enable_add_date_to_title

変換後の小説のタイトルに最新話掲載日や更新日等の日付を付加する

default.title_date_format

enable_add_date_to_title で付与する日付のフォーマット。書式は http://bit.ly/1m5e3w7 を参照

default.title_date_align

enable_add_date_to_title で付与する日付の位置

default.title_date_target

enable_add_date_to_title で付与する日付の種類。
general_lastup(最新話掲載日),last_update(更新日),new_arrivals_date(新着を確認した日),convert(変換した日)

default.enable_ruby_youon_to_big

ルビの拗音(ぁ、ぃ等)を商業書籍のように大きくする

default.enable_pack_blank_line

縦書きで読みやすいように空行を減らす

default.enable_kana_ni_to_kanji_ni

漢字の二と間違えてカタカナのニを使っていそうなのを、漢字に直す

default.enable_insert_word_separator

単語選択がしやすいように単語単位の区切りデータを挿入する（Kindle専用）

default.enable_insert_char_separator

文字選択がしやすいように１文字ずつ区切りデータを挿入する（Kindle専用。enable_insert_word_separator が有効な場合無この設定は無視される）

default.enable_strip_decoration_tag

HTMLの装飾系タグを削除する（主にArcadiaの作品に影響）
